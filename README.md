# Telecom Customer Churn Analysis

This project analyzes customer churn for a telecom company using Excel.  
Key insights include churn rate by segment, top churn contracts, revenue per customer, and senior customer analysis.  

## How to Use
Open the `Telecom_Churn_Analysis.xlsx` file to explore dashboards, charts, and pivot tables.  

## Key KPIs
- Total Customers
- Churn Rate
- Retention Rate
- Top Churn Segment
- Avg Revenue Per Customer
- CLTV
